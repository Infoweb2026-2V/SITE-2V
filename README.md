Repositório oficial do portfólio da **Turma 2V (1º Ano)** do curso técnico integrado em **Informática para Internet** do IFRN - Campus Caicó. Este projeto foi desenvolvido para servir como vitrine dos nossos projetos, aprendizados e contatos.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído puramente com tecnologias base da web (Vanilla), sem o uso de bibliotecas ou frameworks pesados, garantindo alta performance e controle total do código:

* **HTML5:** Estruturação semântica de todo o conteúdo.
* **CSS3:** * Uso intensivo de variáveis (Custom Properties) para manter a paleta de cores estrita.
  * Layouts responsivos construídos com Flexbox e CSS Grid.
  * Efeitos avançados como `drop-shadow`, `backdrop-filter` e transições de estado (`hover`, `transform`).
* **JavaScript (Vanilla):** Lógica de interatividade, manipulação do DOM e controle de tempo (`setTimeout`) para animações visuais.
